# @extence/react-native-appmetrica
React Native wrapper around AppMetrica
## Installation

```sh
npm install @extence/react-native-appmetrica
```

## Usage

```js
import { multiply } from "@extence/react-native-appmetrica";

// ...

const result = await multiply(3, 7);
```
